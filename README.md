# HUD发射端

HUD发射端是一个车机端导航卡片截图发送器。

它会使用 Android MediaProjection 屏幕捕获能力，从车机导航界面中裁切指定区域，并通过 WiFi 局域网 HTTP POST 发送到 HUD接收端显示。

## 主要功能

- 输入接收端 IP 和端口
- 自定义裁切 X / Y / 宽度 / 高度
- 通过截图选择裁剪区域
- 设置发送间隔和 JPG 质量
- 前台服务持续发送
- 关于软件说明

## 使用建议

1. 车机端与接收端建议连接同一个 WiFi 或手机热点。
2. 建议给接收端设置静态 IP，避免每次上车 IP 变化。
3. 如果导航卡片裁切不准，可以手动修改 X/Y/宽度/高度，或使用“通过截图选择裁剪区域”。
4. 第一版推荐 1 秒 1 帧、JPG质量70。
5. 本软件只在局域网内发送裁剪后的导航画面，不上传数据到互联网。
