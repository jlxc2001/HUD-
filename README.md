# HUD发射端

车机端 App：截取导航软件左上角导航信息卡片，并通过 WiFi 局域网发送到 HUD接收端。

## 主要功能

- 软件名：HUD发射端
- 支持手动填写接收端 IP、端口、裁切 X/Y/宽度/高度、发送间隔、JPG质量
- 支持“通过截图选择裁剪区域”
- 支持自动模式：检测到高德地图车机版 `com.autonavi.amapauto` 在前台时发送
- 支持停止发送后通知接收端清空最后一帧
- 支持两类快捷方式：
  - Android 8+ 静态快捷方式：长按 App 图标可见
  - 旧版/自动化软件快捷方式：可在 MacroDroid 的“快捷方式”列表中选择“开启HUD服务 / 关闭HUD服务”

## 快捷方式说明

- **开启HUD服务**：打开后直接跳转屏幕捕获授权界面，适合配合 MacroDroid 模拟点击“立即开始”。
- **关闭HUD服务**：立即停止录屏服务，并向接收端发送停止指令，使接收端清空导航画面，只保留时间和 GPS 速度等 HUD 信息。

## 自动模式说明

Android 不能在完全无感的情况下开始录屏，仍然需要用户授权一次屏幕捕获。

自动模式的实际逻辑是：

1. 打开 HUD发射端，点击开始发送或运行“开启HUD服务”快捷方式；
2. 授权屏幕捕获；
3. 发射端后台运行；
4. 当检测到目标导航包名在前台时发送；
5. 如果系统无法返回前台包名，会继续发送，避免画面冻结。

## 使用建议

1. 建议车机端和接收端连接同一个 WiFi 或手机热点。
2. 建议给接收端设置静态 IP，避免每次上车 IP 变化。
3. 推荐 1 秒 1 帧、JPG质量70。
4. 如果导航卡片裁切不准，使用“通过截图选择裁剪区域”重新选择。
