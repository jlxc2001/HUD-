# HUD发射端

车机端导航卡片截图发送器。用于将高德地图车机版左上角导航信息卡片裁切后，通过 WiFi 局域网发送给 HUD接收端。

## 主要功能

- 手动填写接收端 IP、端口、裁切 X/Y/宽/高、发送间隔、JPG质量。
- “通过截图选择裁剪区域”：支持横屏截图选择，手指拖动或按钮微调裁切框。
- 自动检测导航软件前台：默认包名 `com.autonavi.amapauto`。
- 支持 Android 快捷方式：
  - 开启HUD服务：直接跳转屏幕捕获授权，适合配合 MacroDroid 自动点击授权。
  - 关闭HUD服务：立即停止录屏，并通知接收端清空导航画面。
- 停止发送时会向接收端发送停止指令，接收端会自动黑屏，仅保留车速表和时间信息。

## 使用建议

1. 建议车机端和接收端连接同一个 WiFi 或手机热点。
2. 建议给接收端设置静态 IP，避免每次上车 IP 变化。
3. Android 不允许普通 App 静默开始录屏，首次启动仍需用户确认屏幕捕获授权。
4. 可以用 MacroDroid 启动“开启HUD服务”快捷方式，再模拟点击授权弹窗，实现接近一键启动。
